# Blog API Node.js

A starter REST API for blog posts built with Node.js and Express.

## Features
- Create posts
- List posts
- Get a single post
- Delete posts

## Getting Started
1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the API:
   ```bash
   npm start
   ```
3. Server runs at `http://localhost:3200`
